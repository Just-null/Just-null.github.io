Nginx安装

发表于2021-04-09更新于2021-04-09Nginx
字数总计:432|阅读时长:1分钟|阅读量:

准备工作

使用远程工具连接Linux服务器

下载nginx

网址：http://nginx.org/en/download.html

安装nginx的相关依赖

安装pcre

网址：https://ftp.pcre.org/pub/pcre/

下载（这里我选择安装8.44）

将下载好的pcre传输到Linux服务器

解压pcre

输入tar -xvf pcre-8.44.tar.gz解压pcre

进入解压之后的pcre文件夹

输入./configure

使用make && make install命令对pcre编译并且安装

输入pcre-config --version检查安装的版本号

安装其他依赖

使用yum -y install make zlib zlib-devel gcc-c++ libtool opssl opssl-devel命令安装其他依赖

安装nginx

将nginx文件传输到Linux服务器

输入tar -xvf nginx-1.18.0.tar.gz解压nginx

进入解压后的nginx文件夹

输入./configure

使用make && make install命令对nginx编译并且安装

进入/usr/local文件夹，找到nginx文件夹

进入nginx文件夹下的sbin文件夹

输入./nginx启动nginx

访问nginx（默认端口为80），直接输入服务器地址即可访问

解决问题

如果访问不到，比如下面这种情况

有两种解决方案

  • 关闭防火墙

    • 输入systemctl stop firewalld.service命令关闭防火墙

  • 开放访问的端口号，80 端口

    • 查看开放的端口：firewall-cmd --list-all

    并没有80端口

    • 设置开放的端口号（80端口）
      • firewall-cmd --zone=public --add-port=80/tcp --permanent

    • 重启防火墙
      • systemctl restart firewalld.service

    • 再次查看端口号

再次访问

Nginx常用命令

  • 使用nginx操作命令的前提条件：必须先进入nginx目录
    • cd /usr/local/nginx/sbin

查看nginx版本号

命令：./nginx -v

启动nginx

命令：./nginx

关闭nginx

命令：./nginx -s stop

重加载nginx

命令：./nginx -s reload
