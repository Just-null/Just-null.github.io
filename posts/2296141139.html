简介

  • any()函数的参数是一个可迭代对象，其中的一个元素有一个为真，则any()函数返回真，只有全部为假的时候才返回假。

注：元素除了 0、空、None、False 外都算为真。

语法

any(iterable)

参数

iterable—元组(tuple)、列表(list)，字典(dict)

返回值

True或者False

参数

iterable—元组(tuple)、列表(list)，字典(dict)

range

简介

  • range函数可以创建一个整数列表，在Python3中，range() 函数返回的是一个可迭代对象，而不是列表类型。

语法

参数说明

  • start: 计数从 start 开始。默认是从 0 开始。例如range（5）等价于range（0， 5）;
  • stop: 计数到 stop 结束，但不包括 stop。例如：range（0， 5） 是[0, 1, 2, 3, 4]没有5
  • step：步长，默认为1。例如：range（0， 5） 等价于 range(0, 5, 1)

filter

简介

  • filter() 函数用于过滤序列，过滤掉不符合条件的元素，返回由符合条件元素组成的新列表。
  • 该接收两个参数，第一个为函数，第二个为序列，序列的每个元素作为参数传递给函数进行判断，然后返回 True 或 False，最后将返回 True 的元素放到新列表中。

注：*Pyhton2.7 返回列表，Python3.x 返回迭代器对象*

参数

  • function – 判断函数。
