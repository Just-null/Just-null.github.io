  • #{}иЎЁзӨәдёҖдёӘеҚ дҪҚз¬ҰеҸ·

йҖҡиҝҮ#{}еҸҜд»Ҙе®һзҺ° preparedStatementеҗ‘еҚ дҪҚз¬Ұдёӯи®ҫзҪ®еҖјпјҢиҮӘеҠЁиҝӣиЎҢ java зұ»еһӢе’Ң jdbc зұ»еһӢиҪ¬жҚўпјҢ#{}еҸҜд»Ҙжңүж•ҲйҳІжӯў sql жіЁе…Ҙ,иҝҷж ·еҒҡжӣҙе®үе…ЁпјҢжӣҙиҝ…йҖҹпјҢйҖҡеёёд№ҹжҳҜйҰ–йҖүеҒҡжі•гҖӮ

#{}еҸҜд»ҘжҺҘж”¶з®ҖеҚ•зұ»еһӢеҖјжҲ– pojo еұһжҖ§еҖјгҖӮ еҰӮжһң parameterType дј иҫ“еҚ•дёӘз®ҖеҚ•зұ»еһӢеҖјпјҢ#{}жӢ¬еҸ·дёӯеҸҜд»ҘжҳҜ value жҲ–е…¶е®ғеҗҚз§°гҖӮ

  • ${}иЎЁзӨәжӢјжҺҘsql дёІ

йҖҡиҝҮ${}еҸҜд»Ҙе°Ҷ parameterType дј е…Ҙзҡ„еҶ…е®№жӢјжҺҘеңЁ sql дёӯдё”дёҚиҝӣиЎҢ jdbc зұ»еһӢиҪ¬жҚўпјҢ ${}еҸҜд»ҘжҺҘж”¶з®Җ

еҚ•зұ»еһӢеҖјжҲ– pojo еұһжҖ§еҖјпјҢеҰӮжһң parameterType дј иҫ“еҚ•дёӘз®ҖеҚ•зұ»еһӢеҖјпјҢ${}жӢ¬еҸ·дёӯеҸӘиғҪжҳҜ value,дҪҝз”Ё Statement жҠҠ sql иҜӯеҸҘе’Ң${}зҡ„еҶ…е®№иҝһжҺҘиө·жқҘгҖӮдё»иҰҒз”ЁеңЁжӣҝжҚўиЎЁеҗҚпјҢеҲ—еҗҚпјҢдёҚеҗҢеҲ—жҺ’еәҸзӯүж“ҚдҪңгҖӮ
