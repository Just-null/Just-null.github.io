mybatis#和$的区别

发表于2020-07-21更新于2020-07-21
字数总计:368|阅读时长:1分钟|阅读量:
  • #{}表示一个占位符号

通过#{}可以实现preparedStatement向占位符中设置值，自动进行 java 类型和 jdbc 类型转换，#{}可以有效防止 sql 注入,这样做更安全，更迅速，通常也是首选做法。

#{}可以接收简单类型值或 pojo 属性值。 如果 parameterType 传输单个简单类型值，#{}括号中可以是 value或其它名称。

  • ${}表示拼接sql 串

通过${}可以将 parameterType 传入的内容拼接在 sql 中且不进行 jdbc 类型转换，${}可以接收简单类型值或 pojo 属性值，如果 parameterType 传输单个简单类型值，${}括号中只能是 value,使用 Statement 把 sql 语句和${}的内容连接起来。主要用在替换表名，列名，不同列排序等操作。
