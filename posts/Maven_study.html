Maven жҰӮиҝ°

  • Maven жҳҜ Apache иҪҜд»¶еҹәйҮ‘дјҡз»„з»Үз»ҙжҠӨзҡ„дёҖж¬ҫиҮӘеҠЁеҢ–жһ„е»әе·Ҙе…·пјҢдё“жіЁжңҚеҠЎдәҺ Java е№іеҸ°зҡ„йЎ№зӣ®жһ„е»әе’Ңдҫқиө–з®ЎзҗҶгҖӮMaven иҝҷдёӘеҚ•иҜҚзҡ„жң¬ж„ҸжҳҜпјҡдё“е®¶пјҢеҶ…иЎҢгҖӮиҜ»йҹіжҳҜ[вҖҳmeЙӘv(Йҷ)n]жҲ–[вҖҳmevn]гҖӮ
  • Maven жҳҜзӣ®еүҚжңҖжөҒиЎҢзҡ„иҮӘеҠЁеҢ–жһ„е»әе·Ҙе…·пјҢеҜ№дәҺз”ҹдә§зҺҜеўғдёӢеӨҡжЎҶжһ¶гҖҒеӨҡжЁЎеқ—ж•ҙеҗҲејҖеҸ‘жңүйҮҚиҰҒдҪңз”ЁпјҢMaven жҳҜдёҖж¬ҫеңЁеӨ§еһӢйЎ№зӣ®ејҖеҸ‘иҝҮзЁӢдёӯдёҚеҸҜжҲ–зјәзҡ„йҮҚиҰҒе·Ҙе…·гҖӮ
  • Maven еҸҜд»Ҙж•ҙеҗҲеӨҡдёӘйЎ№зӣ®д№Ӣй—ҙзҡ„еј•з”Ёе…ізі»пјҢжҲ‘д»¬еҸҜд»Ҙж №жҚ®дёҡеҠЎе’ҢеҲҶеұӮйңҖиҰҒд»»ж„ҸжӢҶеҲҶдёҖдёӘйЎ№зӣ®гҖӮ
  • Maven жҸҗдҫӣи§„иҢғзҡ„з®ЎзҗҶеҗ„дёӘеёёз”Ё jar еҢ…еҸҠе…¶еҗ„дёӘзүҲжң¬пјҢе№¶дё”еҸҜд»ҘиҮӘеҠЁдёӢиҪҪе’Ңеј•е…ҘйЎ№зӣ®дёӯгҖӮ
  • Maven еҸҜд»Ҙж №жҚ®жҢҮе®ҡзүҲжң¬иҮӘеҠЁи§ЈеҶі jar еҢ…зүҲжң¬е…је®№й—®йўҳгҖӮ
  • Maven еҸҜд»ҘжҠҠ jar еҢ…жүҖдҫқиө–зҡ„е…¶е®ғ jar еҢ…иҮӘеҠЁдёӢиҪҪе№¶еј•е…ҘйЎ№зӣ®гҖӮ

еҚі Project Object Model йЎ№зӣ®еҜ№иұЎжЁЎеһӢгҖӮMaven жҠҠдёҖдёӘйЎ№зӣ®зҡ„з»“жһ„е’ҢеҶ…е®№жҠҪиұЎжҲҗдёҖдёӘжЁЎеһӢпјҢеңЁ xml ж–Үд»¶дёӯ
иҝӣиЎҢеЈ°жҳҺпјҢд»Ҙж–№дҫҝиҝӣиЎҢжһ„е»әе’ҢжҸҸиҝ°пјҢpom.xml жҳҜ Maven зҡ„зҒөйӯӮгҖӮжүҖд»ҘпјҢmaven зҺҜеўғжҗӯе»әеҘҪд№ӢеҗҺпјҢжүҖжңүзҡ„еӯҰд№ е’Ң
ж“ҚдҪңйғҪжҳҜе…ідәҺ pom.xml зҡ„гҖӮ

  • еҹәжң¬дҝЎжҒҜ

    groupId з»„з»Ү idпјҢдёҖиҲ¬жҳҜе…¬еҸёеҹҹеҗҚзҡ„еҖ’еҶҷгҖӮ ж јејҸеҸҜд»Ҙдёәпјҡ

    еҹҹеҗҚеҖ’еҶҷгҖӮ дҫӢеҰӮ com.baidu

    artifactId йЎ№зӣ®еҗҚз§°пјҢд№ҹжҳҜжЁЎеқ—еҗҚз§°пјҢеҜ№еә” groupId дёӯ йЎ№зӣ®дёӯзҡ„еӯҗйЎ№зӣ®гҖӮ
    version йЎ№зӣ®зҡ„зүҲжң¬еҸ·гҖӮеҰӮжһңйЎ№зӣ®иҝҳеңЁејҖеҸ‘дёӯпјҢжҳҜдёҚзЁіе®ҡзүҲжң¬пјҢйҖҡеёёеңЁзүҲжң¬еҗҺеёҰ-SNAPSHOTversion дҪҝз”ЁдёүдҪҚж•°еӯ—ж ҮиҜҶпјҢдҫӢеҰӮ 1.1.0гҖӮ
    packaging йЎ№зӣ®жү“еҢ…зҡ„зұ»еһӢпјҢеҸҜд»ҘдҪҝ jarгҖҒwarгҖҒrarгҖҒearгҖҒpomпјҢй»ҳи®ӨжҳҜ jar

  • дҫқиө–
    dependencies е’Ңdependency
    Maven зҡ„дёҖдёӘйҮҚиҰҒдҪңз”Ёе°ұжҳҜз®ЎзҗҶ jar еҢ…пјҢдёәдәҶдёҖдёӘйЎ№зӣ®еҸҜд»Ҙжһ„е»әжҲ–иҝҗиЎҢпјҢйЎ№зӣ®дёӯдёҚеҸҜйҒҝе…Қзҡ„пјҢдјҡдҫқиө–еҫҲеӨҡе…¶д»–зҡ„ jar еҢ…пјҢеңЁ Maven дёӯпјҢиҝҷдәӣ jar е°ұиў«з§°дёәдҫқиө–пјҢдҪҝз”Ёж Үзӯҫ dependency жқҘй…ҚзҪ®гҖӮиҖҢиҝҷз§Қдҫқиө–зҡ„й…ҚзҪ®жӯЈжҳҜйҖҡиҝҮеқҗж ҮжқҘе®ҡдҪҚзҡ„пјҢз”ұжӯӨжҲ‘д»¬д№ҹдёҚйҡҫзңӢеҮәпјҢmaven жҠҠжүҖжңүзҡ„ jar еҢ…д№ҹйғҪи§ҶдёәйЎ№зӣ®еӯҳеңЁдәҶгҖӮ

  • й…ҚзҪ®еұһжҖ§
    properties properties жҳҜ з”Ё жқҘ е®ҡ д№ү дёҖ дәӣ й…Қ зҪ® еұһ жҖ§ зҡ„ пјҢ дҫӢ еҰӮproject.build.sourceEncodingпјҲйЎ№зӣ®жһ„е»әжәҗз Ғзј–з Ғж–№ејҸпјүпјҢеҸҜд»Ҙи®ҫзҪ®дёәUTF-8пјҢйҳІжӯўдёӯж–Үд№ұз ҒпјҢд№ҹеҸҜе®ҡд№үзӣёе…іжһ„е»әзүҲжң¬еҸ·пјҢдҫҝдәҺж—ҘеҗҺз»ҹдёҖеҚҮзә§гҖӮ

  • жһ„е»ә

  • з»§жүҝ
    parent еңЁ Maven дёӯпјҢеҰӮжһңеӨҡдёӘжЁЎеқ—йғҪйңҖиҰҒеЈ°жҳҺзӣёеҗҢзҡ„й…ҚзҪ®пјҢдҫӢеҰӮпјҡgroupIdгҖҒversionгҖҒжңүзӣёеҗҢзҡ„дҫқиө–гҖҒжҲ–иҖ…зӣёеҗҢзҡ„з»„д»¶й…ҚзҪ®зӯүпјҢд№ҹжңүзұ»дјј Java зҡ„з»§жүҝжңәеҲ¶пјҢз”Ё parent еЈ°жҳҺиҰҒз»§жүҝзҡ„зҲ¶е·ҘзЁӢзҡ„ pom й…ҚзҪ®гҖӮ

  • иҒҡеҗҲ
    modules еңЁ Maven зҡ„еӨҡжЁЎеқ—ејҖеҸ‘дёӯпјҢдёәдәҶз»ҹдёҖжһ„е»әж•ҙдёӘйЎ№зӣ®зҡ„жүҖжңүжЁЎеқ—пјҢеҸҜд»ҘжҸҗдҫӣдёҖдёӘйўқеӨ–зҡ„жЁЎеқ—пјҢиҜҘжЁЎеқ—жү“еҢ…ж–№ејҸдёә pomпјҢе№¶дё”еңЁе…¶дёӯдҪҝз”Ё modules иҒҡеҗҲзҡ„е…¶е®ғжЁЎеқ—пјҢиҝҷж ·йҖҡиҝҮжң¬жЁЎеқ—е°ұеҸҜд»ҘдёҖй”®иҮӘеҠЁиҜҶеҲ«жЁЎеқ—й—ҙзҡ„дҫқиө–е…ізі»жқҘжһ„е»әжүҖжңүжЁЎеқ—пјҢеҸ« Maven зҡ„иҒҡеҗҲгҖӮ

еқҗж Ү(g3av)

Maven жҠҠд»»дҪ•дёҖдёӘжҸ’д»¶йғҪдҪңдёәд»“еә“дёӯзҡ„дёҖдёӘйЎ№зӣ®иҝӣиЎҢз®ЎзҗҶпјҢз”ЁдёҖз»„(дёүдёӘ)еҗ‘йҮҸз»„жҲҗзҡ„еқҗж ҮжқҘиЎЁзӨәгҖӮеқҗж ҮеңЁд»“еә“
дёӯеҸҜд»Ҙе”ҜдёҖе®ҡдҪҚдёҖдёӘ Maven йЎ№зӣ®гҖӮ

artifactIdпјҡжЁЎеқ—еҗҚпјҢйҖҡеёёжҳҜе·ҘзЁӢеҗҚ

versionпјҡзүҲжң¬еҸ·

йңҖиҰҒзү№еҲ«жҢҮеҮәзҡ„жҳҜпјҢйЎ№зӣ®еңЁд»“еә“дёӯзҡ„дҪҚзҪ®жҳҜз”ұеқҗж ҮжқҘеҶіе®ҡзҡ„пјҡgroupIdгҖҒartifactId е’Ң version еҶіе®ҡйЎ№зӣ®еңЁд»“еә“дёӯзҡ„и·Ҝеҫ„пјҢartifactId е’Ң version еҶіе®ҡ jar еҢ…зҡ„еҗҚз§°гҖӮ

дҫқиө–(dependency)
