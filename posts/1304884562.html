WindowsзҺҜеўғдёӢgcc/g++зҡ„е®үиЈ…еҸҠй…ҚзҪ®

еҸ‘иЎЁдәҺ2020-04-12|жӣҙж–°дәҺ2020-04-12|ж•ҷзЁӢgcc/g++Windows
|еӯ—ж•°жҖ»и®Ў:676|йҳ…иҜ»ж—¶й•ҝ:2еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

д»Җд№ҲжҳҜgcc/g++

GCC:GNU Compiler Collection(GUN зј–иҜ‘еҷЁйӣҶеҗҲ)пјҢе®ғеҸҜд»Ҙзј–иҜ‘CгҖҒC++гҖҒJAVгҖҒFortranгҖҒPascalгҖҒObject-CгҖҒAdaзӯүиҜӯиЁҖгҖӮ

gccжҳҜGCCдёӯзҡ„GUN C CompilerпјҲC зј–иҜ‘еҷЁпјү

g++жҳҜGCCдёӯзҡ„GUN C++ CompilerпјҲC++зј–иҜ‘еҷЁпјү

д»Җд№ҲжҳҜMinGW

MinGW жҸҗдҫӣдәҶдёҖеҘ—з®ҖеҚ•ж–№дҫҝзҡ„WindowsдёӢзҡ„еҹәдәҺGCC зЁӢеәҸејҖеҸ‘зҺҜеўғгҖӮMinGW ж”¶йӣҶдәҶдёҖзі»еҲ—е…Қиҙ№зҡ„Windows дҪҝз”Ёзҡ„еӨҙж–Үд»¶е’Ңеә“ж–Үд»¶пјӣеҗҢж—¶ж•ҙеҗҲдәҶGNU ( http://www.gnu.org/ )зҡ„е·Ҙе…·йӣҶпјҢзү№еҲ«жҳҜGNU зЁӢеәҸејҖеҸ‘е·Ҙе…·пјҢеҰӮз»Ҹе…ёgcc, g++, makeзӯүгҖӮMinGWжҳҜе®Ңе…Ёе…Қиҙ№зҡ„иҮӘз”ұиҪҜд»¶пјҢе®ғеңЁWindowsе№іеҸ°дёҠжЁЎжӢҹдәҶLinuxдёӢGCCзҡ„ејҖеҸ‘зҺҜеўғпјҢдёәC++зҡ„и·Ёе№іеҸ°ејҖеҸ‘жҸҗдҫӣдәҶиүҜеҘҪеҹәзЎҖж”ҜжҢҒпјҢдёәдәҶеңЁWindowsдёӢе·ҘдҪңзҡ„зЁӢеәҸе‘ҳзҶҹжӮүLinuxдёӢзҡ„C++е·ҘзЁӢз»„з»ҮжҸҗдҫӣдәҶжқЎд»¶гҖӮ
еҺҹж–Үй“ҫжҺҘпјҡhttps://blog.csdn.net/jpcfei/article/details/6428613

MinGWе’ҢMinGW-w64зҡ„еҢәеҲ«

MinGW-w64 дёҺ MinGW зҡ„еҢәеҲ«еңЁдәҺ MinGW еҸӘиғҪзј–иҜ‘з”ҹжҲҗ32дҪҚеҸҜжү§иЎҢзЁӢеәҸпјҢиҖҢ MinGW-w64 еҲҷеҸҜд»Ҙзј–иҜ‘з”ҹжҲҗ 64дҪҚ жҲ– 32дҪҚ еҸҜжү§иЎҢзЁӢеәҸгҖӮMinGW зҺ°е·Іиў« MinGW-w64 жүҖеҸ–д»Ј

дёәд»Җд№ҲдҪҝз”ЁMinGW-w64

  1. MinGW-w64 жҳҜејҖжәҗиҪҜд»¶пјҢеҸҜд»Ҙе…Қиҙ№дҪҝз”ЁгҖӮ
  2. MinGW-w64 з”ұдёҖдёӘжҙ»и·ғзҡ„ејҖжәҗзӨҫеҢәеңЁжҢҒз»ӯз»ҙжҠӨпјҢеӣ жӯӨдёҚдјҡиҝҮж—¶гҖӮ
  3. MinGW-w64 ж”ҜжҢҒжңҖж–°зҡ„ CиҜӯиЁҖ ж ҮеҮҶгҖӮ
  4. MinGW-w64 дҪҝз”Ё Windows зҡ„CиҜӯиЁҖиҝҗиЎҢеә“пјҢеӣ жӯӨзј–иҜ‘еҮәзҡ„зЁӢеәҸдёҚйңҖиҰҒз¬¬дёүж–№ DLL пјҢеҸҜд»ҘзӣҙжҺҘеңЁ Windows дёӢиҝҗиЎҢгҖӮ
  5. йӮЈдәӣи‘—еҗҚзҡ„ејҖжәҗ IDE е®һйҷ…еҸӘжҳҜе°Ҷ MinGW-w64 е°ҒиЈ…дәҶиө·жқҘпјҢдҪҝе®ғжӢҘжңүеҸӢеҘҪзҡ„еӣҫеҪўеҢ–з•ҢйқўпјҢз®ҖеҢ–дәҶж“ҚдҪңпјҢдҪҶеҶ…йғЁж ёеҝғд»Қз„¶жҳҜ MinGW-w64гҖӮ

еӣ жӯӨпјҢеҰӮжһңйңҖиҰҒеңЁWindowsзі»з»ҹдёӢдҪҝз”Ёgcc/g++зӯүејҖеҸ‘е·Ҙе…·пјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮе®үиЈ…mingwзҡ„ж–№ејҸжқҘиҺ·еҫ—гҖӮ
